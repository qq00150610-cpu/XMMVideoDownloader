# XMM Video Downloader 📱🎬

X/Twitter Video Downloader Android App

## Features

- 🔗 **X Link Parsing** - Paste any X/Twitter video link to extract HD MP4
- ▶️ **Online Playback** - Stream videos directly with ExoPlayer
- 📥 **Video Download** - Save videos locally with progress notifications
- 📂 **Local Videos** - Browse and play downloaded videos
- 📱 **Fullscreen** - Immersive landscape/portrait playback
- 🔒 **Lock Screen** - Keep screen on during playback
- ⏩ **Fast Forward/Rewind** - Double-tap left/right ±10 seconds
- 👆 **Gesture Controls** - Swipe left for brightness, right for volume
- ⏭ **Next/Previous** - Playlist navigation
- 🌍 **7 Languages** - EN, ZH, JA, KO, ES, FR, DE
- 🎨 **8 Theme Colors** - Blue, Purple, Green, Orange, Red, Teal, Pink, Indigo

## Build

```bash
# Open in Android Studio and sync, or:
./gradlew assembleDebug
```
